# Sound Effects Directory

This directory contains game sound effects.

## Nuclear Explosion Sounds

To add nuclear explosion sound effects:

1. Download free sound effects from:
   - [Freesound.org - Nuclear Explosion](https://freesound.org/people/ryansnook/sounds/110114/)
   - [Freesound.org - Huge Explosion](https://freesound.org/people/FusionWolf3740/sounds/568876/)
   - [Pixabay - Nuclear Bomb Sounds](https://pixabay.com/sound-effects/search/nuclear%20bomb/)
   - [Uppbeat - Nuclear Explosion SFX](https://uppbeat.io/sfx/category/explosions/nuclear-explosion)

2. Save the sound files as:
   - `nuclear-explosion.mp3` - Main nuclear blast sound
   - `explosion-shockwave.mp3` - Shockwave/rumble sound
   - `explosion-blast.mp3` - Initial blast sound

3. These will automatically play when bombs detonate in the game.
